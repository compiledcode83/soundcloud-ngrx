# SoundCloud NgRx

A basic SoundCloud API client built with Angular2 and NgRx. Try the [live demo](https://soundcloud-ngrx.herokuapp.com).


## Stack
- Angular2
- NgRx Effects
- NgRx Store
- RxJS
- Immutable
- Ava
- Circle CI
- Express
- Heroku
- Karma
- Typescript
- Webpack
